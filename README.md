# Custommade Backend

Backend service for a custom-made tailoring platform built with Spring Boot.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## Features (MVP)
- User roles: Customer, Tailor, Admin
- Tailoring requests & quotes
- Order & payment flow (escrow-based)
- Dispute handling

## How to Run
1. Install Java 17
2. Install PostgreSQL
3. Update `application.properties` with DB credentials
4. Run `CustommadeApplication`

## Status
MVP backend in active development.
